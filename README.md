# 阴阳师壁纸下载器

## 功能说明

本工具是一个专门用于下载阴阳师官方壁纸的 GUI 应用程序，具有以下功能：

### 核心功能
- **图片分类**：支持横版、竖版、手机壁纸三种分类
- **分辨率选择**：每种分类对应多个分辨率选项
  - 横版默认选择 1920x1080 分辨率
  - 竖版默认选择 1080x1920 分辨率
- **目录管理**：
  - 自动检测目录格式
  - 自动创建符合格式的子目录（如 `横版_1920x1080`）
  - 目录切换时同步更新界面显示的保存路径
- **智能排序**：按日期降序、序号升序排列图片
- **下载管理**：
  - 双进度条显示（总下载进度和当前图片下载进度）
  - 支持批量下载（默认每批 10 张）
  - 跳过已存在的文件，避免重复下载
  - 下载过程中禁用其他操作，防止误操作
  - 优雅的停止机制，点击停止后会完成当前批次下载
- **用户界面**：
  - 基于 tkinter 的美观图形界面
  - 支持中文显示
  - 窗口大小可拉伸但有最小限制（320x480）
  - 实时状态更新，显示下载详情

## 系统要求
- Python 3.10+
- tkinter 库（通常随 Python 安装）
- 第三方依赖：
  - requests
  - beautifulsoup4

## 安装依赖
```bash
pip install requests beautifulsoup4
```

## 运行方式
```bash
python download_yys_images_ui.py
```

## 使用说明
1. **选择分类**：在“选择图片分类”区域选择横版、竖版或手机壁纸
2. **选择分辨率**：在“选择分辨率”下拉菜单中选择对应分类的分辨率
3. **选择保存目录**：点击“浏览”按钮选择保存目录
4. **开始下载**：点击“开始下载”按钮开始下载壁纸
5. **查看进度**：通过进度条查看下载进度，通过状态文本框查看详细信息
6. **停止下载**：点击“停止下载”按钮停止下载
7. **关闭窗口**：关闭窗口时会自动等待当前下载完成
